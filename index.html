University Coursework

City Generation for Final year Project

Final year project that generates a city which considers important factors such as terrain to generate a believable European city.

  • Unity, C#
  • In progress
  • Currently generates city in one scene and terrain in the other
  • City is generated via L-Systems
  • Landscape is generated via noise
  • Generated without launching Unity’s “play mode” and automatically re-generates if any of the generation properties were edited

Physics Engine

First part of the coursework for the Gaming Simulations module. Created physics engine that has movement based on the Newtonian Dynamics (forces, velocities) and colision detection (sphere/sphere, sphere/box, box/box).

  • C++
  • Collision response via calcullation of the impulse
  • Optimisation by using broadphase and narrow phase for collision resolution
  • Additional optimisation by transforming each ground and floor tile into a one large physical object with a collider
  • Second part with AI and Pathfinding is in progress

Volcano Eruption Scene

Developed as coursework for the Graphics for Games module. Uses multiple features to create a good looking alien planet.

  • Unity, C#
  • Custom moving shaders for lava textures
  • Handmade volcano model in Blender3D
  • Use of particle systems for smoke, lava bubbles and burning ashes
  • Use of Unity’s terrain tools for terrain
  • Optimisation techniques to ensure stable 60fps
  • Use of camera and scenes for smooth transit between environments

Personal Projects

KillerWave

This game was developed in one week for a Brackeys Game Jam 2021.1. It is a simple game for two players where players must survive countless enemy waves by destroying them with a line which is created only when players are close to each other.

  • Unity, C#
  • Developed in one week
  • Simple graphics inspired by vaporwave and chillwave
  • 6 different enemy types
  • Settings – music, sfx volume
