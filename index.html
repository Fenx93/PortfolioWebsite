University Coursework

City Generation - Final year Project

Final year project that generates a city which considers important factors such as terrain to generate a believable European city.

  • Unity, C#
  • Submitted and got high grades for it
  • City is generated with L-Systems
  • Landscape is generated using noise
  • Created without launching Unity’s “play mode” and automatically re-generates if any of the generation properties were edited

Physics Engine

Coursework for a Gaming Simulations module. Created physics engine that has movement based on the Newtonian Dynamics (forces, velocities) and colision detection (sphere/sphere, sphere/box, box/box).

  • C++
  • Collision response via impulse calculation
  • Collision resolutions are optimised with a broadphase and narrowphase
  • Introduced additional tricks to optimize performance even more
  • A* Pathfinding
  • AI state machine
  • Flocking
  • Playable!

Volcano Eruption Scene

Developed as coursework for the Graphics for Games module. Uses multiple features to create a nicely looking alien planet.

  • Unity, C#
  • Custom moving shaders for lava textures
  • Handmade volcano model in Blender3D
  • Use of particle systems for smoke, lava bubbles and burning ashes
  • Use of Unity’s terrain tools for terrain
  • Optimisation techniques to ensure stable 60fps
  • Use of camera and scenes for smooth transit between environments
