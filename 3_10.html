This is a prototype for a text RPG, heavily inspired by “Reigns“. The player has to constantly choose between two options to progress the story, fulfill various achievemnts and survive for as long as possible. All the art is drawn by myself.

Everything is easily scalable, so can be easily adopted for any platform.

It is heavily inspired by slavic folk tales, with characters like Koschei the Deathless, Firebird, Leshy and Baba Yaga. Of course, some adaptation will be required, for example, changing russian “tsarevna” to “princess”.

The game already has all the mechanics implemented, only story needs finishing touches.
